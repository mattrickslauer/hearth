═══ 1 · HOOK  [camera — slow, warm]
Every AI you've seen this year lives in the same three boxes. A chat window, a code editor, a slide deck.
Meanwhile the real world — the door, the pump, the greenhouse, the nursery — still runs on dumb thresholds somebody hard-coded in 2009.
So we gave it a Qwen brain. Five dollars of hardware. No code. This is Hearth.


═══ 2 · ANY HARDWARE  [hardware, then dashboard — brisk, pointing]
Hearth isn't a product with a parts list — it's a runtime.
One command flashes a board you already own. It boots, and announces itself: "I have a temperature pin. I have a relay."
No drivers. No datasheet. And the hub? A Pi, an old laptop — whatever's in your drawer.
So this is my home, on one screen. That green dot — "live"? Real stream, off the hub in my house.


═══ 3 · THE ASK  [dashboard — brisk]
Now the good part. I don't write code — I just say what I want.
"If the nursery goes over seventy-eight, cut the heater and text me."
I hit Author… and Qwen Cloud compiles that one sentence into a real, running script. There it is — over seventy-eight, heater off, notify me.
I never touched a line of it.
And see that "local" tag? Qwen wrote it — but it runs on the hub, in my house. Cut the internet, it still fires.


═══ 4 · THE PROOF  [hardware + screen — slowest, let it breathe]
So let's make it real. I warm this sensor with my hand… and the tile climbs — seventy-six… seventy-eight… seventy-nine —
the watch flashes, and the feed lights up: Fired… Actuated… Notified.
And — [phone buzzes] —
                          …there's my phone.
Real sensor. Real script. Real alert. No cloud required.


═══ 5 · CLOSE  [wiring flash, then camera — land it]
Don't know how to wire it? Just ask. "Add a soil sensor." Qwen tells you which pin goes where — then writes the code that reads it.
That's the whole idea. Any hardware. Any brain — swap the model, it's yours. Open, from the sensor to the cloud.
Qwen doesn't have to win the chat window. It can win the physical world instead — because it's the one you can afford to put inside things.
This isn't the finished product. It's the platform you build it on.
Clone it… and go build.
